# User Guide

## English Instructions

Welcome to the WiFi-coin wallet! This guide will help you get started with using the wallet effectively.

### Installation
1. **Download** the wallet from the official repository.
2. **Install** the application by following the on-screen instructions.

### Setting Up
1. **Create Account**: Open the application and create your account by filling in the required details.
2. **Backup** your wallet by noting down your recovery phrase.

### Using the Wallet
- **Send Coins**: To send coins, input the recipient's address and the amount.
- **Receive Coins**: Share your wallet address with others to receive coins.

## বাংলা নির্দেশাবলী

ওয়াইফাই-মুদ্রা ওয়ালেটে স্বাগতম! এই গাইডটি আপনাকে ওয়ালেটটি কার্যকরভাবে ব্যবহারে সাহায্য করবে।

### ইনস্টলেশন
1. **পুনঃউল্লেখ করুন** অফিসিয়াল রিপোজিটরি থেকে ওয়ালেটটি ডাউনলোড করুন। 
2. **ইনস্টল করুন** স্ক্রীনে নির্দেশনা অনুসরণ করে।  

### সেটিং আপ
1. **অ্যাকাউন্ট তৈরি করুন**: অ্যাপ্লিকেশনটি খুলুন এবং আপনার তথ্য পূরণ করে অ্যাকাউন্ট তৈরি করুন। 
2. **আপনার ওয়ালেট ব্যাকআপ করুন** আপনার পুনরুদ্ধার বিশ্রাম লিখে। 

### ওয়ালেট ব্যবহার করা
- **কয়েন পাঠান**: কয়েন পাঠাতে, প্রাপকের ঠিকানা এবং পরিমাণ লিখুন।
- **কয়েন গ্রহণ করুন**: অন্যদের সাথে আপনার ওয়ালেট ঠিকানা শেয়ার করুন কয়েন গ্রহণ করার জন্য।
